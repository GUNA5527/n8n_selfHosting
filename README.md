1. Download Docker Desktop from the official website.
2. Choose your operating system (Windows or Mac) and download the installer.
3. Run the installer file.
4. Accept the terms and proceed with default settings.
5. (Windows only) Enable and install WSL 2 when prompted.
6. Wait for the installation to complete.
7. Restart your system if required.
8. Launch Docker Desktop and complete the setup.
