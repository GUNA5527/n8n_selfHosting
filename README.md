step1: Download Docker Desktop from the official website.
step2: Choose your OS (Windows/Mac) and get the installer.
step3: Run the downloaded installer file.
step4: Accept terms and continue with default settings.
step5: (Windows) Allow installation of WSL 2 when prompted.
step6: Wait for the installation to complete.
step7: Restart your computer if asked.
step8: Open Docker Desktop and let it finish setup.
