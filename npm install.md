1.npm Installation Steps
2.Download Node.js from the official website (https://nodejs.org).
3.Choose the recommended LTS version for your operating system.
4.Download the installer file.
5.Run the installer.
6.Accept the license agreement.
7.Click “Next” and proceed with the default settings.
8.Make sure the option “Add to PATH” is selected.
9.Click “Install” and wait for the installation to complete.
10.Click “Finish” after installation completes.
11.Restart your system if required.
12.Open Command Prompt.
13.Verify the installation by running:node -v , npm -v
